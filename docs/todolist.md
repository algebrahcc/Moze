## 已完成

### 基础设施

- Nuxt 4 + Tailwind + shadcn-vue 基础 UI 能力
- Supabase Auth 登录/注册/魔法链接（页面与跳转）
- Quiet Finance 视觉基调（灰蓝/石墨 + 静默交互）
- Iconfont SVG Sprite 图标体系（通过 `AppIcon`）

### 页面（基础视觉/结构）

- 登录页：左右分栏 + 极简品牌区 + 登录/注册/
- 看板/账户/流水/净值：统一卡片与页面结构（视觉层）
- 看板摘要：账户数、本月收支、近 30 天净值条数

### MVP 功能（已落地）

- 账户：新建/编辑/归档/恢复（Supabase accounts）
- 流水：新增交易 + 最近 50 条列表（Supabase transactions）
- 转账：双账户（转出/转入）建模与录入（to_account_id）
- 分类：两级分类表（categories）+ 流水选择与创建（category_id）
- 净值：录入快照 + 最近记录 + 当日盈亏/累计盈亏计算 + 近 90 天趋势（Sparkline）
- 登录：忘记密码邮件发送（Supabase reset）
- 看板：净资产/今日盈亏/资产趋势（基于真实数据聚合）
- 看板：账户构成（现金/信用/股票）
- 账户：余额/净值聚合 + 删除保护（有流水不可删）
- 流水：交易编辑/删除
- 流水：转账视图双栏 + 账户筛选
- 报表：月度/年度收支 + 分类占比
- 报表：月度预算设置与使用进度
- 标签体系：交易标签与筛选
- 净值：盈亏批量回填 + 30/90 区间切换
- 分类：管理页（增删改/层级调整）
- 看板：收支趋势 + 分类构成图表
- 设置：主题切换（亮/暗/跟随系统）
- 设置：全局快速记账入口与交易弹窗联动
- 设置：用户下拉菜单、个人信息页与设置模块
- 设置：迁移分类管理至设置页
- 基础设施：全局 Toast 通知系统与预算/同步异常提示
- 导航：净值页更名为“财富”
- 资产：支持分组展示（日常/投资/固定/负债）与负债显性化
- 总览：待办事项（信用卡还款提醒/未设置提醒）
- 流水：日历视图与批量删除

## 待完成

### 阶段一：补齐短板 (短期 1-3 个月)

> 目标：降低用户迁移门槛，完善核心记账体验。

- **数据导入/导出**
    - 支持标准 CSV 导入（适配钱迹/随手记格式）
    - 支持全量数据导出备份
- **周期性交易**
    - 支持设置每月/每周自动记账（如房租、订阅费）
- **PWA 体验优化**
    - 优化移动端触摸交互
    - 支持离线记账（Service Worker）
- **搜索功能**
    - 支持按金额、备注、分类搜索历史交易

### 阶段二：增强分析 (中期 3-6 个月)

> 目标：提升数据价值，辅助财务决策。

- **高级预算**
    - 支持分类预算（如“餐饮”限额 2000）
    - 支持预算超支提醒
- **资产负债表**
    - 生成标准的月度/年度资产负债表，清晰展示财务健康度
- **多币种支持**
    - 基础汇率换算
    - 资产折合本币展示
- **标签管理增强**
    - 支持标签分组
    - 标签报表分析

### 阶段三：智能化与生态 (长期 6-12 个月)

> 目标：降低记账成本，拓展使用场景。

- **AI 记账助手**
    - 接入 LLM，支持“昨天打车花了 35”自然语言记账
- **API 开放**
    - 提供 REST API
    - 支持 iOS 快捷指令自动记账
- **多用户协作**
    - 支持家庭账本
    - 权限管理（只读/读写）
- **投资组合追踪**
    - 对接股票/基金 API，自动更新持仓净值
