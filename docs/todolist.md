# Moze（默资）功能 Todo

## 已完成

### 基础设施

- Nuxt 4 + Tailwind + shadcn-vue 基础 UI 能力
- Supabase Auth 登录/注册/魔法链接（页面与跳转）
- Quiet Finance 视觉基调（灰蓝/石墨 + 静默交互）
- Iconfont SVG Sprite 图标体系（通过 `AppIcon`）

### 页面（基础视觉/结构）

- 登录页：左右分栏 + 极简品牌区 + 登录/注册/魔法链接切换
- 看板/账户/流水/净值：统一卡片与页面结构（视觉层）
- 看板摘要：账户数、本月收支、近 30 天净值条数

### MVP 功能（已落地）

- 账户：新建/编辑/归档/恢复（Supabase accounts）
- 流水：新增交易 + 最近 50 条列表（Supabase transactions）
- 转账：双账户（转出/转入）建模与录入（to_account_id）
- 分类：两级分类表（categories）+ 流水选择与创建（category_id）
- 净值：录入快照 + 最近记录 + 当日盈亏/累计盈亏计算 + 近 90 天趋势（Sparkline）
- 登录：忘记密码邮件发送（Supabase reset）
- 看板：净资产/今日盈亏/资产趋势（基于真实数据聚合）
- 看板：账户构成（现金/信用/股票）
- 账户：余额/净值聚合 + 删除保护（有流水不可删）
- 流水：交易编辑/删除
- 流水：转账视图双栏 + 账户筛选
- 报表：月度/年度收支 + 分类占比
- 报表：月度预算设置与使用进度
- 标签体系：交易标签与筛选
- 净值：盈亏批量回填 + 30/90 区间切换
- 分类：管理页（增删改/层级调整）
- 看板：收支趋势 + 分类构成图表
